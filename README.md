# icu4c Puppet Module for Boxen

[![Build Status](https://travis-ci.org/boxen/puppet-icu4c.png?branch=master)](https://travis-ci.org/boxen/puppet-icu4c)

## Usage

```puppet
include icu4c
```

## Required Puppet Modules

* boxen
* homebrew
* stdlib

## Developing

Write code.

Run `script/cibuild`.
